## 0. Introduce

- 차량에서 발생하는 데이터를 OBD 기기로 수집하여 차량 스캐너 앱으로 정보를 전송합니다.

- 이후 시스템이 위험 운전 행동의 전조증상을 분석하고, 위험 운전 행동을 예측한 다음, 그 결과를 앱으로 전송하여 운전자에게 알람을 제공하는 것을 목표로 프로젝트를 수행하였습니다.
